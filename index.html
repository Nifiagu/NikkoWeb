About Me

I’m Nikko M. Puno, a passionate and motivated student of Bachelor of Science in Mechatronics at Cebu Technological University – Danao Campus. I have basic skills in web design and development using HTML and CSS, and I’m familiar with programming microcontrollers using the Arduino IDE and Visual Studio Code. During our thesis project, my classmates and I worked together on a miniaturized packaging system, where we all contributed to the overall design and agreed on the core idea. While I only have basic knowledge in PLC programming, I’m willing to start again from the fundamentals to improve in that area.

For my second project using the ESP8266, I personally worked on the entire web code for the interface, while my groupmates helped by sharing ideas for the design and layout. These experiences taught me the value of collaboration, hands-on learning, and pushing my skills forward. My goal is to continue growing in my course and become more confident and capable in areas like automation, sensor systems, web-based controls, and embedded systems.

My first project is a research-based automation system that highlights the use of sensors, PLC programming, and mechanical components. My second project features an ESP8266 Wi-Fi module that connects sensors to a web interface. It displays sensor data like distance, temperature, and allows real-time interaction with control buttons. The webpage is styled with glowing animations, responsive layouts, and interactive elements.
