About Me

I’am a passionate and motivated student of Bachelor of Science in Mechatronics at Cebu Technological University – Danao Campus. I have basic skills in web design and development using HTML and CSS, and I’m familiar with programming microcontrollers using the Arduino IDE and Visual Studio Code. During our thesis project, my classmates and I worked together on a miniaturized packaging system, where we all contributed to the overall design and agreed on the core idea. While I only have basic knowledge in PLC programming, I’m willing to start again from the fundamentals to improve in that area.

For my second project using the ESP8266, I personally worked on the entire web code for the interface, while my groupmates helped by sharing ideas for the design and layout. These experiences taught me the value of collaboration, hands-on learning, and pushing my skills forward. My goal is to continue growing in my course and become more confident and capable in areas like automation, sensor systems, web-based controls, and embedded systems.

My Projects

Research/Thesis Project

The Miniaturized Factory Packaging Model enhances student learning in industrial automation by offering a compact, hands-on simulation of real packaging systems. Using PLCs, it allows students to design, program, and troubleshoot automation tasks, bridging the gap between theory and practical application in a more engaging, industry-relevant way.

ESP8266 Web Server

This project uses an ESP8266 microcontroller to create a web server that shows real-time distance from an ultrasonic sensor. The data is displayed on a custom HTML page with a glowing design, animated background, and colorful buttons. It blends IoT, sensors, and web development into an interactive Mechatronics application.
